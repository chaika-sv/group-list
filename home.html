List

Быстрый старт

Скачайте файлы:

  • jquery.list.js
  • jquery.list.css
  • jQuery откуда-нибудь

И подключите их в своём проекте:

			
<link rel="stylesheet" href="css/jquery.list.css">
<script src="js/jquery-3.1.1.min.js"></script>
<script src="js/jquery.list.js"></script>

HTML разметка (предварительно сортировать необязательно):

			
<div id="contacts-html" class="list">
  <ul>
    <li>Владилен Ягупов</li> 
    <li>Игорь Чернаков</li> 
    <li>Алина Аргамакова</li> 
    ...
  </ul>
</div>

Пример настройки списка:

			
$("#contacts-controls").list(
  {
    withControls: 1,  //  Используем элементы управления
    compareBy: 2  //  Сортировка по второму слову
  }
);

Опции

compareBy

Тип: Number

По-умолчанию: 2

Выбор элемента для сортировки

withControls

Тип: Number

По-умолчанию: 0

Установите 1 – использовать элементы управления, 0 – не использовать

items

Тип: Array

По-умолчанию: []

Массив элементов списка

Пример:

			
$("#contacts").list(
{
  compareBy: 1,  //  сортируем по первому слову
  items: [       //  элементы
    'Экель Максим', 
    'Шабунин Алексей', 
    'Беленков Семен', 
    ...
  ],
  withControls: 1   //  используем элементы управления
});

Методы

AddNewItem

Параметр: String

По-умолчанию: нет

Добавление элемента в список

RemoveItem

Параметр: String

По-умолчанию: нет

Удаление элемента из списка

Filter

Параметр: String

По-умолчанию: нет

Применить фильтр

ChangeSorting

Параметр: Number

По-умолчанию: нет

Поменять элемент для сортировки

Пример:

			
$("#contacts").list.AddNewItem('Алиева Светлана');  //  добавляем элемент
$("#contacts").list.RemoveItem('Баринова Анна');  //  удаляем элемент
$("#contacts").list.Filter('а');  // фильтруем
$("#contacts").list.ChangeSorting(2);  //  меняем элемент для сортировки

Формирование списка JS-параметром

Ранее было показано как задать список HTML разметкой.

Второй способ – создаём пустой HTML-контейнер (можно и не пустой – элементы заданные обоими способовами "сложатся" в один список):

			
<div id="contacts-js-param" class="list"></div>

JavaScript (передаём массив items):

			
$("#contacts-js-param").list(
{
  compareBy: 1,
  items: [
    'Экель Максим', 
    'Шабунин Алексей', 
    'Беленков Семен', 
    ...
  ]
});

Элементы управления

Можно настроить элементы управления:

  • добавление элемента
  • удаление элемента
  • вариант группировки
  • фильтр

Для начала установите значение withControls:

			
$("#contacts-controls").list(
{
  withControls: 1   //  используем элементы управления
});

Добавление элемента

Добавьте контейнер с классом newItem и аттрибутом list: ID_вашего_списка. Внутри разместите contenteditable с классом newItemInput и ссылку с классом newItemBtn.

					
<div class="newItem" list="contacts-controls">
	<div class="newItemInput" contenteditable="true"></div>
	<a class="newItemBtn" href="#">Добавить</a>
</div>

Удаление элемента

Для удаления элементов достаточно установить значение withControls – теперь при наведении курсора на элемент списка появится крестик для его удаления.

Группировка

Добавьте контейнер с классом sorting и аттрибутом list: ID_вашего_списка. Внутри разместите select с классом sortingSelect и опциями со значения 1 и 2.

							
<div class="sorting" list="contacts-controls">
	<div class="sortingLabel">Сортировка</div>
	<select class="sortingSelect">
		<option value=1>Первое слово</option>
		<option value=2>Второе слово</option>
	</select>
</div>

Фильтр

Добавьте контейнер с классом filter и аттрибутом list: ID_вашего_списка. Внутри разместите contenteditable с классом filterText.

							
<div class="filter" list="contacts-controls">
	<div class="filterLabel">Фильтр</div>
	<div class="filterText" contenteditable="true"></div>
</div>
